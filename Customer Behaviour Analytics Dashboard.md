# 📊 Customer Behaviour Analysis Tool

An interactive front-end project that simulates customer financial behaviour analysis, inspired by real-world banking and fintech data analytics systems.

---

## 🚀 Overview

This project allows users to explore and segment a dataset of customers based on:
- Account balance
- Activity level
- Risk profile

It demonstrates how raw data can be transformed into actionable insights using JavaScript logic, similar to SQL-style analysis.

---

## 🧠 Key Features

- View all customers in a clean dashboard
- Filter high-value customers (high balance)
- Identify high-risk customers
- Dynamic data rendering using JavaScript
- Simple and responsive UI design

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)

## 📊 What I Learned

## 💡 Business Insight Simulation

This project simulates a basic banking analytics system where:
- High-value customers can be identified for retention strategies
- High-risk customers can be flagged for review
- Customer segmentation helps support decision-making
